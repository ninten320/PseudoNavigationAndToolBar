# PseudoNavigationAndToolBar

擬似NavigationBarとToolbarのサンプル

* コードによる端末やバージョンの判定は不要
* StoryboardでSafe Areaからバーのコンテンツ部分の高さを指定する
* UINavigationBarやUIToolBarのようにSafe Areaの外側にも同じ背景色がつく

## 環境

* Xcode 9.0
* Swift 4.0
